# MedVision AI - Machine Learning Thesis
Computer-aided diagnosis of mammogram images using CNN+CLAHE, ResNet50, VGG16, and VGG19. Includes RAG-enhanced pathology reports.

## Features
- Mammogram analysis with multiple AI models
- Explainable AI (Grad-CAM)
- RAG-powered clinical report generation
- Animated Dashboard and Analysis History
